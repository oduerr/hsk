Help

Session Controls

  • New Run – Start a fresh session on the currently selected level.
  • Replay… – List your recorded sessions, replay only mistaken cards, resume checkpoints.
  • Save Progress – Create a checkpoint you can resume later.

Settings (Gear)

  • Level – Pick HSK 0–6 or combine custom levels. Levels are discovered automatically from data/hsk*.csv. Audio files are stored in data/recordings/.
  • Auto‑reveal – Enable and set seconds; a countdown shows at the bottom.
  • Minimal UI – Hide Back/Reveal/Next (use gestures/keys).
  • Outdoor – High‑contrast theme; Light mode is also available.
  • Audio feedback – Beeps when marking/unmarking mistakes.
  • Voice – Browser TTS or OpenAI TTS (model, voice, speed, cache). Test and compare quickly.
  • Info – Version, build time, checkpoint/session info; Export/Import/Undo.

Gestures & Shortcuts

  • Swipe left/right: Next/Back
  • Tap Mistake button; on mobile, tap anywhere on the card to toggle mistake
  • Space / ↑ / ↓: Reveal / Unreveal
  • Enter / →: Next; ←: Back; M: Mistake; R: Replay

Tone Visualizer

  • Open with the mic (🎙️) button above the card.
  • See live pitch, a final smoothed contour, and a recorded spectrogram.
  • Use Re‑record / Stop / Replay controls in the dialog header.
